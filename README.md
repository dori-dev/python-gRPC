# Python gRPC

Implementation of greetings in Python with gRPC

#

## How to Run

<br>
Install requirements

```
pip install -r requirements.txt
```

<br>
Run greet server

```
python greet_server.py
```

<br>
Run greet client

```
python greet_client.py
```
